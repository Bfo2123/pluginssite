About Someones Plugins?

Someones Plugins was created to provide free, easy to use, and very helpful plugins for the community. It was also created to help Someones Son? learn about pluginc creation. He wanted to make plugins accessable by everyone, which includes easy setup and config. On most of his plugins, you can configure everything from a GUI in game, which helps new minecraft server owners setup their server!

Get it Now!

Easy Setup and Configuration

To setup one of Someones Sons? Plugins, simply go to the spigot page, which can be found above, and download the latest version of whatever plugin it is. Then, drag it into your plugins folder. Once its in your plugins folder, simply restart the server. For JoinLeaveMessages, you can simply go in game and run /jlconfig and set everything up. For GamemodeSwapper, go to the config.yml file, and you can configure it there, then reload the plugin by running /gsr in game! If you need any assistance, join our Discord!
